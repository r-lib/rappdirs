# rappdirs: Application Directories: Determine Where to Save Data, Caches, and Logs

An easy way to determine which directories on the users computer you
should use to save data, caches and logs. A port of Python's 'Appdirs'
(<https://github.com/ActiveState/appdirs>) to R.

## Main functions

- [`user_data_dir()`](https://rappdirs.r-lib.org/dev/reference/user_data_dir.md)

- [`user_config_dir()`](https://rappdirs.r-lib.org/dev/reference/user_data_dir.md)

- [`user_cache_dir()`](https://rappdirs.r-lib.org/dev/reference/user_cache_dir.md)

- [`site_data_dir()`](https://rappdirs.r-lib.org/dev/reference/site_data_dir.md)

- [`user_log_dir()`](https://rappdirs.r-lib.org/dev/reference/user_log_dir.md)
